# SQL Exploratory Data Analysis (EDA) Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) using SQL on a sales database.
The analysis includes database exploration, dimension analysis, measures analysis,
and ranking analysis to extract business insights.

## Database
- Fact Table: gold.fact_sales
- Dimension Tables: gold.dim_customers, gold.dim_products

## Analysis Performed
- Database and column exploration
- Dimension analysis (customers, products)
- Date analysis
- Measures analysis (sales, quantity, orders)
- Magnitude analysis
- Ranking analysis

## SQL Concepts Used
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- Aggregate functions
- JOIN
- UNION ALL
- Window functions
